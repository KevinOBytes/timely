# Potential Features & Widgets for Billabled

1. **Global Command Palette (CMD + K)**
   - A universal search bar that opens seamlessly anywhere in the app to jump between projects, tasks, invoices, or users via keyboard.

2. **Client Approval Workflows & Stripe Connect**
   - Expanding the client portal to allow clients to "Sign Off" on weekly timesheets and immediately pay generated Invoices via a Stripe secure checkout session.

3. **Retainer & Budget Burndown Dashboards**
   - A new visualization widget for projects that operate on monthly retainers or fixed budgets to track pace and prevent over-servicing.

4. **Idle Time Detection & Focus Modes**
   - Detects system inactivity and asks the user whether to discard idle time or keep the timer running. Can bundle with a Pomodoro Widget for 25-minute sprints.

5. **Expense Tracking with Receipt OCR**
   - A widget to upload receipt images where AI parses the total/date and appends reimbursable expenses to projects and invoices.

6. **Public API & Personal Access Tokens**
   - Allow users to generate API keys to interact with Billabled programmatically, opening up native integrations (Zapier, Make, custom CRMs).

7. **Custom Role / Permissions Matrix**
   - Allow admins to create granular custom roles with explicit boolean toggles (e.g., Accountant role: can see Invoices, but cannot see Project Tasks).

8. **Time Tracking Widget (Desktop/Menu Bar app)**
   - A native desktop widget or browser extension to start/stop timers without opening the full web portal.

9. **Multi-Currency Support**
   - Billing clients in different currencies and handling exchange rates seamlessly within the invoice generator.

10. **Public API & Webhook Debugger**
    - An interface for developers to test webhooks and review raw JSON dispatch logs.
